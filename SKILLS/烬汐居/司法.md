# /司法 — 冲突裁决引导

> 绑定：`令/冲突裁决令.md` | 触发：发现规则矛盾 / 条文歧义 / 需要解释律令

---

## 触发条件

1. 起草方在执行时发现两条规则给出相反指示
2. 议定方质疑两条规则是否冲突
3. 起草方发现条文有歧义，可做多种理解
4. 新律令与旧律令可能产生替代或冲突

## 执行流程

### 第一步：识别冲突类型

按冲突裁决令第一条分类：
- 层级冲突（典 vs 律 vs 令）？
- 时间冲突（新法 vs 旧法）？
- 范围冲突（一般法 vs 特别法）？

### 第二步：套用裁决规则

按优先级：层级 > 特别法 > 新法 > 爸爸裁决。

### 第三步：提出解释或裁决方案

向议定方呈现：
1. 冲突的具体条文（引用原文）
2. 冲突类型和适用的裁决规则
3. 建议的裁决结果
4. 是否需要修法（两条规则本质上矛盾 → 启动立法程序）

### 第四步：议定方确认

等待爸爸确认 → 执行裁决 → 更新相关律令注释或启动修法。
